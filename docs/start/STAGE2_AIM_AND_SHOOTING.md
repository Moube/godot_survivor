# 阶段2：瞄准与射击完成记录

## 阶段目标

本阶段的目标是建立最小可验收的射击闭环：

1. 玩家始终朝向鼠标方向
2. 点击或按住射击输入时可以稳定发射子弹
3. 子弹按目标方向飞行
4. 子弹不会无限留在场景中
5. 子弹撞到墙体或障碍物时销毁

## 本阶段完成内容

### 1. 玩家朝向逻辑完成

`scene/player/Player.cs` 已补充鼠标瞄准逻辑：

- 在 `_PhysicsProcess()` 中持续读取鼠标世界坐标
- 根据玩家位置到鼠标位置的方向计算旋转角度
- 让玩家根节点始终朝向鼠标

同时修正了 `Player.tscn` 中朝向标记与代码朝向定义不一致的问题：

- `FacingMarker` 现在以本地 `+X` 作为前方
- `Muzzle` 发射点也放置在本地 `+X` 前方

这保证了视觉朝向、枪口方向和射击方向一致。

### 2. 玩家射击逻辑完成

`scene/player/Player.cs` 已实现最小射击逻辑：

- 导出 `BulletScene`
- 导出 `FireCooldownSeconds`
- 在玩家场景中读取 `Muzzle` 节点作为发射点
- 响应 `shoot` 输入实例化 `Bullet.tscn`
- 设置子弹生成位置与飞行方向
- 使用冷却时间限制连续射击频率

当前实现支持按住鼠标左键持续射击。

### 3. 子弹场景基础结构完成

`scene/bullet/Bullet.tscn` 已完成最小可视化与碰撞配置：

- 根节点使用 `Area2D`
- 添加 `CollisionShape2D`
- 添加临时 `Polygon2D` 作为子弹外形

当前子弹外观是临时占位图形，后续如果接入正式素材，可以直接替换显示节点。

### 4. 子弹飞行与生命周期完成

`scene/bullet/Bullet.cs` 已实现：

- `Speed` 作为飞行速度
- `LifetimeSeconds` 作为生命周期时长
- `Initialize(Vector2 direction)` 用于初始化飞行方向
- `_PhysicsProcess()` 中按方向持续移动
- 生命周期结束后自动销毁

因此当前子弹具备稳定飞行和自动清理能力，不会无限堆积在场景中。

### 5. 子弹撞墙销毁完成

`scene/bullet/Bullet.cs` 已补充碰撞后销毁逻辑：

- 在 `_Ready()` 中订阅 `BodyEntered`
- 当子弹进入墙体、边界或障碍物这类 `PhysicsBody2D` 时调用 `QueueFree()`

由于 `Level01.tscn` 中当前墙体与障碍物使用的是 `StaticBody2D`，因此这部分逻辑已经可以直接生效。

## 验收结果

本阶段已完成并通过以下验收项：

1. 玩家朝向标记会正确朝向鼠标
2. 子弹从玩家前方枪口位置发射
3. 子弹沿鼠标方向飞行
4. 子弹在生命周期结束后自动销毁
5. 子弹撞到墙体或障碍物后立即销毁
